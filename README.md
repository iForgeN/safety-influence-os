# Safety Influence OS

**A practical web tool for Occupational Health & Safety influence and accountability.**

## Origin
Evolved from insights on OHS power dynamics and stakeholder alignment.

## Core Modules
- **Stakeholder Incentive Map**
- **Dual-Statement Communication**
- **Frontline Signal Log**
- **Advice & Accountability Register**

## Tech
PostgreSQL backend with React frontend planned.

See `docs/schema.md` for database design.
